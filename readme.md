## CAN インターフェイスまとめ
| CAN ID        | Tx/Rx | byte長 | Data                                                                                   |
|---------------|-------|-------|-------------------------------------------------------------------------------------|
| 0x011 | Tx    | 1    | [0:バッテリー電流超過、非常停止要求(0xff):1] |
| 0x001 | Rx    | 1    | [0:非常停止命令(0xff):1] |
| 0x201 | Tx    | 4     | (double)[0:電流値(A):3]                                                                   |
